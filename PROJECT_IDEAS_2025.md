- Focus on fundamental concepts
- Use popular, well-documented technologies
- Aim for projects completable in 1-4 weeks

### For Intermediate (⭐⭐⭐)

- Include user interface and backend components

### For Advanced (⭐⭐⭐⭐⭐)
